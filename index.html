Hay 4 maneras para trabajar con CSS:

Vamos a ver de menos recomendado a más recomendado:

  • Añadiendo un atributo style sobre el elemento HTML y definiendo las reglas CSS.
  • Desde dentro del documento HTML con style dentro del HEAD del documento
  • Desde dentro del documento HTML con style dentro del HEAD del documento pero con import url
  • Desde un fichero externo CSS que asociamos a un HTML mediante a link rel="stylesheet" href="style.css"
